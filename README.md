# Customer Segmentation with RFM Analysis

This project focuses on customer segmentation using RFM (Recency, Frequency, Monetary) analysis with Python.

## 📌 Project Overview

The goal is to analyze customer behavior and group customers based on their purchasing patterns.

## 📊 Dataset

* Superstore Sales Dataset
* Includes order date, customer ID, sales, and order information

## ⚙️ Methodology

* Data cleaning and preprocessing
* RFM metric calculation:

  * Recency: Days since last purchase
  * Frequency: Number of orders
  * Monetary: Total spending
* Scoring customers into 4 groups using quantiles
* Segment creation based on RFM scores

## 📈 Key Insights

* Champions: High-value and frequent customers
* Loyal Customers: Consistent and engaged buyers
* At Risk: Customers who have not purchased recently
* Lost Customers: Inactive customers

## 🛠️ Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## 📂 Output

* RFM scores
* Customer segments
* Visual analysis

## 🚀 Next Steps

* Churn analysis
* Machine learning models for prediction

## 💡 Business Insights & Actions

* Champions customers generate the highest revenue → focus on retention strategies  
* At Risk customers show declining activity → targeted campaigns are required  
* Lost customers have high churn probability → reactivation campaigns can be tested  
* New customers should be nurtured to increase frequency
